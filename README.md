# LIT Enterprises

Central brand website for **LIT Enterprises** — the experiential heart of the TGG family, igniting inner sparks across sport, lifestyle, media & culture, and wellness.

## Stack
- Static HTML / CSS / JS (no build step)
- Fonts: Libre Baskerville + Inter (Google Fonts)
- Deployed via **Cloudflare Pages** (Git-native), matching the TGG-web infrastructure

## Pages
- `index.html` — Home ("Be LIT")
- `story.html` — Our Story (spark philosophy + TGG family)
- `ecosystem.html` — Ecosystem (4 pillars, all ventures)
- `moments.html` — Moments gallery (filter + lightbox)
- `sparks.html` — Sparks Ignited (impact stories)
- `join.html` — Join / Partner
- `contact.html` — Contact
